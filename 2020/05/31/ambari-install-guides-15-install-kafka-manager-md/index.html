1、简介

Kafka Manager 可用于监控和管理 kafka 集群。

主要作用有：

  • 便捷的监控集群状态（topics，consumers，offsets，brokers，副本分布，分区分布）；
  • 根据自定义配置创建 topic；
  • 删除 topic，但是前提是配置 kafka 的 delete.topic.enable=true；
  • 增加已存在 topic 的分区；
  • 更新已存在 topic 的配置等等。
